## Ingredients:
- **300mL** (frozen) mangoes
- **100mL** (frozen) pineapple
- **Half can** Sprite Zero (adjust for viscosity)
- **1 tbps** cane sugar
- **3 tbps** plain yogurt

## Instructions:
- Partially thaw fruit if frozen
- Blend ingredients in blender
- Enjoy
